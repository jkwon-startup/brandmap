# 여행가J의 브랜드맵 워크시트

질문에 답하면 **나만의 브랜드맵**이 완성되는 단일 HTML 웹앱입니다.
6단계(PART) 질문에 답을 채우면, 완성된 브랜드맵을 **마크다운 · 텍스트 · PDF**로 내려받을 수 있습니다.

- **라이브**: https://jkwon-startup.github.io/brandmap/
- 설치 · 로그인 불필요. 작성 내용은 브라우저에 자동 저장됩니다.

## 구성 (6 PART / 19 섹션)

| PART | 내용 |
| --- | --- |
| PART 1. 브랜드 정의 | 브랜드 정수(Purpose/Vision/Mission) · 핵심 가치 · 브랜드 철학 |
| PART 2. 고객 정의 | 핵심 고객(페르소나) · 고객 문제(Pain) · 고객의 할 일(JTBD) |
| PART 3. 시장 정의 | 경쟁 환경 · 포지셔닝 · 신뢰 근거(RTB) |
| PART 4. 브랜드 표현 | 브랜드 개성 · 톤 & 키워드 · 슬로건 · 감각 설계 |
| PART 5. 실행 계획 | 핵심 제품 · 주요 기능 & 지표 · 고객 여정 · 제품 구성 |
| PART 6. 미래 계획 | 핵심 내러티브(PDR) · 브랜드 확장 로드맵 |

## 사용법

1. 브라우저에서 [링크](https://jkwon-startup.github.io/brandmap/)를 엽니다.
2. 각 PART 탭의 질문에 답을 채웁니다. (가이드와 예시가 함께 표시됩니다)
3. 입력은 자동 저장됩니다. 완료되면 하단 **[브랜드맵 저장 (.md)]** 으로 내보내세요.

> ⚠️ 작성 내용은 이 브라우저에만 저장됩니다. 다른 기기에서 이어 쓰거나 보관하려면
> 반드시 다운로드해 두세요. (`[새 브랜드맵]`은 모든 내용을 지웁니다)

## 개발

기술 스택 · 아키텍처 · 배포 절차는 [`DEVELOPMENT.md`](./DEVELOPMENT.md) 참고.
출력 포맷의 기준은 `브랜드맵 최종 출력 예시/` 폴더(로컬 전용)입니다.
